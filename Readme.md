🔍 Overview

This project analyzes loan portfolio data to uncover insights related to risk, customer behavior and data quality. The dashboard is built using Power BI & SQL and focuses on identifying key trends in loan distribution, defaults and delinquency.

🎯 Objectives
Analyze loan distribution across different dimensions (Grade, State, Age, Purpose)
Identify default and delinquency patterns
Highlight data quality issues (missing values)
Provide actionable business recommendations

📁 Dataset Summary
Total Loan Amount: ₹732M
Total Loans: 65K
Total Collection: ₹808M

📌 Key Insights
🔹 Risk Metrics
Default Loans: 1K (1.56%)
Delinquent Loans: 7K (10.84%)

🔹 Data Quality Issues
Missing Grade: 25K loans (~38%)
Missing State: 5K loans

🔹 Customer Segmentation
Age Group 26–35 contributes the highest loan amount

Top Loan Purposes:
Home
Services
Business

🔹 Loan Characteristics
Term Distribution:
60 months → 60%
36 months → 40%

🔹 Portfolio Health
~90% loans are non-default
~80% loans are non-delinquent
Similar trends observed across loan status and verification status

💡 Key Recommendations
1. Improve Data Quality
Make Grade & State mandatory fields
Implement validation checks in data entry systems
2. Strengthen Risk Monitoring
Build early warning systems for delinquency
Monitor high-risk segments closely
3. Target High-Value Segments
Focus marketing on 26–35 age group
Customize loan offerings based on purpose
4. Optimize Loan Tenure
Encourage shorter tenure loans to reduce risk exposure
5. Simplify Verification Process
Re-evaluate verification steps to balance risk and user experience
